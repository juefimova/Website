Music App

This is a music store application built with Kotlin that allows users to search and sort songs. It contains welcoming Start, Login, Sing Up pages, after Login user is redirected on the Main page with music lists and search tool.
For developing this project were used: navigation, databases (Room), dialog windows for various interactions, search tools (AutoComplete), Intent.
